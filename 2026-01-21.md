### 1. Linear Block Coding (LBC)
- for single bit error correction

- for the block of k message bits, n - k redundant check bits are added.
#### 1.1 LBC Encoder
- generation matrix, G = \[ I$_k$ , P$_{key}$]
- k bits of Message, m$_k$ 
- code work, C (n, k) : m$_k$ 
- redundant bits, R$_q$
- R$_q$ = \[ M$_k$]\[P$_{k*q}$]

### 2. Hamming codes
- defined as the n, k linear block codes that statisfies the following conditions:
	- number of check bits, q >= 3
	- block length n = 2$^q$ - 1
	- no. of message bits k = n-q
	- minimum distance d$_{min}$ = 3
	- code rate r = k/n - 1 -q/n = 1 -q/(2$^q$ - 1)
- features
	- encoder
		- even or add parity is added
	- combination is 2$^{q}$ > m + q + 1
- for example, C (7, 4)
- m = 4
- q = 3
- 2$^{3}$ = 4 + 3 + 1
- so, q = 3
- c = m : p
- m = m1 m2 m3 m4 
- p = p1 p2 p4
- in position of 1st, 2nd, 4th
- p1 p2 m1 p4 m2 m3 m4
